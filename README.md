---
## Короткое описание работы
- Для вывода короткой справки
```bash
python -m src.main --help
```
- Команды запускаются по одной с помошью
```bash
python -m src.main <команда для исполнения>
```
- Справку о команде можно получить через
```bash
python -m src.main <команда для исполнения> --help
```


---
## Поддерживаемые команды
- **ls** - Выводит список файлов в текущей директории. Опция *-l* добавляет к выводу информацию о правах доступа, размере, времени последнего изменения
- **cd** - Меняет текущую директорию на указанную
- **cat** - Выводит содержимое файла. *-b* побитовое чтение файла
- **cp** - Копирует файл или папку из одной папки в другую. *-r* рекурсивное копирование файлов внутри папки
- **mv** - Переносит файл или папку из одной папки в другую.
- **rm** - Удаление файла или папки. *-r* рекурсивное удаление файлов внутри папки
- **zip**, **tar** - архивируют папку в формате zip или tar соответственно
- **unzip**, **untar** - Разархивируют архив в одноимённую папку

## Принятые решения
- Поддерживаются относительные и абсолютне пути
- Для расширяемости программы используется импорт по списку файлов из "src/configs/plugins_config.py"
- Для добавления новой команды нужно добавить в конфиг строку с названием модуля, как при стандартном импорте в Python
- Т.к. используется библиотека typer, логируется не пользовательский ввод, а аргументы распознанные typer
- Т.к. используется библиотека typer, команды каждый раз запускаются как новый процесс. Текущая директория сохраняется в ".cwd"
- Логирование ведётся в файл shell.log
